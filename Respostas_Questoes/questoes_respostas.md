##### 5 - O que é a função lambda em Python? Dê um exemplo de uso.
###### R: Função (lambda) é uma função anônima, que não possui um nome, tem como objetivo ser simples e concisa.
```
numeros = [1, 2, 3, 4, 5]
quadrados = list(map(lambda x: x ** 2, numeros))
print(quadrados)  # Saída: [1, 4, 9, 16, 25]
```

##### 6 - Como você ordenaria uma lista de números em ordem decrescente?
###### R: Assim
```
numeros = [5, 2, 8, 1, 9, 3]
numeros.sort(reverse=True)
print(numeros)  # Saída: [9, 8, 5, 3, 2, 1]
```
